> **Installation:**

> - git clone https://github.com/korayzorluoglu/miisAlisveris.git
> - touch database/database.sql
> - composer install
> - php artisan key:generate
> - php artisan migrate
> - php artisan db:seed
> - php artisan currency:update


 **Cron Jobs:**
for  Currencies update from Finance Yahoo;
> - php artisan currency:update
